💻 Java

✔ 2진수, 8진수, 16진수 리터럴

  • 2진수 리터럴은 접두어 0b, 8진수 리터럴은 접두어 0, 16진수 리터럴은 접두어 0x를 붙인다.
    🤔 쓸 일은 별로 없을 것 같은데 배울 때마다 헷갈린다…




✔ BigInteger, BigDecimal

  • long 타입으로 충분하지 않은 정수형 숫자를 표현할 경우 BigInteger 클래스를 사용한다.
  • 반올림 오류가 없는 부동소수점 계산이 필요할 경우 BigDecimal 클래스를 사용한다.




✔ 단락 평가

  • && 연산자 단락 평가: 첫 번째 조건이 false일 경우 두 번째 조건은 평가하지 않는다.
    두 번째 조건이 오류를 일으킬 가능성이 있을 때 유용
  • || 연산자 단락 평가: 첫 번째 조건이 true일 경우 두 번째 조건은 평가하지 않는다.




✔ 서식 지정자

  • 서식 지정자는 % 문자로 시작한다.
변환 문자 목적
d 정수
x 16진수
o 8진수
f 고정 부동소수점
e 지수 부동소수점
g 일반 부동소수점(e와 f중 짧은 형태)
a 16진 부동소수점
s 문자열
c 문자
b boolean
h 해시 코드
t 날짜와 시간
  • 서식 지정 출력의 모양을 제어하는 플래그를 지정할 수도 있다.
플래그 목적
+ 양수와 음수의 부호를 출력한다.
빈칸 양수 앞에 빈칸(스페이스)를 추가한다.
- 필드를 왼쪽 정렬한다.
0 앞에 0을 추가한다.
( 음수 값을 괄호로 감싼다.
, 그룹 분리자를 사용한다. (ex. 1,000 단위 , 추가)
#(f 서식) 항상 소수점을 포함한다.
#(x 또는 o 서식) 접두어 0x 또는 0을 추가한다.
$ 서식을 지정할 인자의 인덱스를 지정한다.
ex. %1$d %1$x는 첫 번째 인자를 10진수와 16진수로 출력
< 앞의 명세와 같은 값의 서식을 지정한다.
ex. %d %<x는 같은 수를 10진수와 16진수로 출력한다.
  • String.format 메서드는 서식이 지정된 문자열을 반환한다.




✔ 인스턴스 변수의 기본값

  • 인스턴스 변수는 항상 명시적으로 초기화해야 하는 지역변수와 달리 기본값이 있다.
  • 숫자는 0, boolean 값은 false, 객체 참조는 null이 기본값으로 설정 된다.




✔ 정적 초기화 블록

  • 정적 초기화는 메모리로 클래스를 처음 불러올 때 일어난다.
public class className {
  ...
  static {
    // 정적 초기화 블록
  }
}




✔ 정적 임포트

import static java.lang.Math.*;
  • 위와 같은 형태의 지시문을 소스 파일의 상단에 추가하면 클래스 이름 접두어를 붙이지 않고도 Math 클래스의 정적 메서드와 정적 변수를 사용할 수 있다.
  • 흔히 java.util.Comparator와 java.util.stream.Collectors를 사용할 때는 정적 임포트 선언을 사용한다.




✔ 중첩 클래스

📃 정적 중첩 클래스

  • 정적 중첩 클래스는 해당 클래스가 외부 클래스에 속해 있다는 사실을 분명하게 해줄 뿐 일반적인 클래스와 거의 차이가 없다.
  • 중첩 클래스의 인스턴스가 외부 클래스의 어느 인스턴스에 속하는지 알 필요가 없을 때 사용한다.

📃 내부 클래스

  • 내부 클래스의 메서드는 외부 클래스(아우터 클래스)의 인스턴스 변수에 접근할 수 있다.
  • 중첩 클래스의 인스턴스가 외부 클래스의 어느 인스턴스에 속하는지가 중요할 때 사용한다.

내부 클래스용 특수 문법 규칙

  • 외부 클래스 참조를 나타내는 문법은 외부클래스명.this로 외부 클래스 참조를 나타낸다.
    이는 필수가 아니며 외부 클래스의 변수나 메서드를 참조하면 암시적으로 외부 클래스 참조를 사용한다.
  • 내부 클래스에는 컴파일 시간 상수 말고는 정적 멤버를 선언할 수 없다.
